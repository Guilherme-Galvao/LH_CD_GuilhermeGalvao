# LH_CD_GuilhermeGalvao
Modelo Preditivo | Precificação | Indicium

Previsão de Preços de Aluguéis em Nova York


Este projeto tem como objetivo analisar e prever os preços de aluguéis de imóveis em Nova York utilizando um conjunto de dados fornecido. Por meio de técnicas de ciência de dados e machine learning, exploraremos as variáveis disponíveis, testaremos hipóteses e desenvolveremos um modelo preditivo eficaz.

Objetivos Principais:
Análise Exploratória de Dados (EDA): Identificar padrões e relações entre as variáveis.
Modelagem Preditiva: O Modelo preve o preço de aluguel com base nos dados fornecidos.
Insights: Responder a perguntas de negócios relacionadas ao mercado imobiliário e escolha de modelo para análise.
Tecnologias e Ferramentas: Linguagem: Python | Bibliotecas principais: pandas, numpy, scikit-learn, matplotlib, seaborn, nltk, joblib.
Ambiente de desenvolvimento: Jupyter Notebook
